# Onboarding-Steps-React
New User Onboaring Steps App.

### Features
+ Sliding Forward/Backward Animation
+ Data Validation on Each Screen
+ Error Handling
+ Firebase Connection to Persist Data

##### Built With
+ HTML5
+ CSS3
+ React
+ Semantic UI

### Getting Started
clone repo

`cd` into folder and run `npm install`

run `npm start`
